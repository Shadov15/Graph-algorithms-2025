# Graph-algorithms-2025
Kurs algorytmów grafowych na Akademii Górniczo-Hutniczej w Krakowie (rocznik 2025).

## Laboratoria
- **Lab 1:** Rozgrzewka
- **Lab 2:** Maksymalny przepływ
- **Lab 3:** Spójność krawędziowa
- **Lab 4:** Dokończenie algorytmów przepływowych
- **Lab 5 & 6:** Grafy przekątniowe
- **Lab 7:** Biblioteka NetworkX, planarność, przepływy i SAT-2CNF

## Projekty
- **Projekt 1:** Szczęśliwe dzielniki
- **Projekt 2:** Inwigilacja
